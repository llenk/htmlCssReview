Priorities

Under the hood, CSS assigns a point value to how relevant things are. Putting styling directly on the element (in html) is first place, then ids, then classes, then type.

Alternately, you can put !important on an attribute in CSS to make it most important.

Order in CSS does matter in a specific case. If you write the same thing with different properties twice, the one that comes last wins.

The important thing is not to memorize the ranking, but remembering that there is a way to be ranked. And accept it.
